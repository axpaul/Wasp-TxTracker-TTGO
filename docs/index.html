📟 Console & Terminal

📋 AT COMMAND CHEATSHEET
AT
Test communication with the station
AT+HELP
Print the complete help menu
AT+INFO
Query station identification
AT+CFG
Display the detailed configuration
AT+ID?
Query tracker ID (SSID Num)
AT+TYPE?
Query tracker type (0=FX, 1=MF, 2=BALLOON, 3=OTHER)
AT+INTERVAL?
Query transmission interval (sec)
AT+FREQ?
Query the active frequency
AT+SF?
Query active Spreading Factor
AT+BW?
Query active Bandwidth
AT+POWER?
Query LoRa TX power (dBm)
AT+CRC?
Query CRC configuration status
AT+DEBUG=1
Enable extended debug logs
AT+DEBUG=0
Disable extended debug logs
AT+BINUSB=1
Enable binary telemetry output
AT+BINUSB=0
Disable binary telemetry output
AT+SAVE
Save config dynamically to NVS
AT+RESET
Factory reset and reboot station
